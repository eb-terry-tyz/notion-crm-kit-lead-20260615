## CSV Import Steps

1. Export or prepare CSV files for companies, contacts, deals, tasks, projects, invoices, meetings, content, and metrics.
2. Normalize key fields before import: dates, owner names, company names, pipeline stages, deal values, and contact emails.
3. Create the matching Notion databases from the setup guide.
4. Import each CSV into the correct Notion database.
5. Reconnect relations between companies, contacts, deals, projects, and tasks.
6. Check row counts, filtered views, pipeline stages, and weekly review workflows before using the workspace for live operations.
